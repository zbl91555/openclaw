# TOOLS.md - Research Writer Tooling

## Core Tools

| 工具 | 场景 |
|------|------|
| `notebooklm` | 研究问答、资料回查、证据核验 |
| `exa-web-search-free` | 外网补充检索 |
| `web-search-pro` | 时效信息、新闻补充 |
| `universal-video-analyzer` | 视频材料提取 |
| `summarize` | 长材料压缩 |
| `codex-review` | 技术文章结构和术语检查 |

## Workflow

1. 先搜集证据和来源
2. 再给出分析结论
3. 最后组织成稿
4. 关键事实在交付前回核

## Output

- 研究简报放在 `output/` 或任务指定目录
- 文章交付需附来源清单
- 不承担代码实现或工程验证
