# Advanced Skill Creator

This skill provides a comprehensive 5-step research flow for creating OpenClaw skills according to official standards and best practices.

## Purpose

The Advanced Skill Creator ensures proper methodology when users request to create or modify OpenClaw/Moltbot/ClawDBot skills. It follows a strict 5-step process to guarantee quality and compliance with official standards.

## Features

- Executes official 5-step research flow
- Consults official documentation
- Researches public skills on ClawHub/ClawdHub
- Searches for best practices
- Performs solution fusion and comparison
- Generates properly structured output

## Steps Performed

1. Consults official documentation from multiple sources
2. Researches related public skills on ClawHub/ClawdHub
3. Searches for best practices and security considerations
4. Fuses solutions and compares across key dimensions
5. Formats output according to required structure

## Usage

When triggered, this skill automatically performs comprehensive research and generates properly structured skills that follow official standards and best practices.
