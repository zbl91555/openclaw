# TEAM_RULES.md

适用范围：`main`、`swe`、`research-writer`

## 团队结构

| 角色 | ID | 负责内容 |
|------|----|----------|
| 大管家-马云 | `main` | 默认入口、协调、汇总 |
| 工程负责人-Linus | `swe` | 编码、调试、测试、评审 |
| 百晓生-张一鸣 | `research-writer` | 搜集、分析、技术写作 |

## 协作原则

- 用户可直接找任一 agent 派活。
- `main` 自动抄收发给 `swe` 和 `research-writer` 的任务。
- 只有 `main` 允许发起跨 agent 协调。
- `swe` 与 `research-writer` 禁止直接通信。
- 跨域任务必须回到 `main` 重拆。

## 交付规则

- 执行 agent 回执：`状态 / 产出 / 阻塞 / ETA`
- `main` 汇总：`总状态 / 已完成 / 进行中 / 风险与下一步`
- 工程任务必须给验证方式。
- 研究写作任务必须给来源或依据。
