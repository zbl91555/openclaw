# TOOLS.md — 笔杆子·工具路由规则

## ✍️ 写作工具链

| 工具 | 场景 |
|------|------|
| `summarize` | 长文本提炼、多文档合并摘要、AI 摘要生成 |
| `notebooklm` | 建立知识库、生成播客脚本和音频 |
| 原生文字生成 | 推文、日报、短内容（无需额外工具） |

## 📖 内容来源

写作前从以下位置读取素材：
- `~/.openclaw/shared/brief.md` — 百晓生的情报简报
- `~/.openclaw/shared/output/` — 代码极客的技术产出
- 飞书群聊中的历史上下文

## 📝 日报模板

```markdown
## 📋 项目日报 · [日期]

### 🔍 今日情报
[来自 brief.md 的核心发现，2-3 条]

### 💻 技术进展
[来自代码极客的产出，一句话说清做了什么]

### ✅ 已完成事项
- [ ]

### 📌 待 Lin 哥决策
[需要确认的内容，列清楚选项]

### 🎯 明日计划
- 百晓生：[任务]
- 代码极客：[任务]
- 笔杆子：[任务]
```

## 📲 推文规范

- 长度：200-500 字，手机屏幕一页内读完
- 结构：金句开头 → 核心内容 → 引导互动
- 风格：通俗、有温度，避免机器腔
